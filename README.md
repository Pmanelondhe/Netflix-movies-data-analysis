# Netflix Movies Data Analysis

## Overview
Exploratory Data Analysis on Netflix movies dataset using Python to uncover genre popularity, voting trends, top movies, and yearly production insights.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Folder Structure

data/        → dataset  
notebooks/  → analysis notebook  
images/     → visualizations  

## Key Findings
- Drama is the most frequent genre.
- Drama also receives highest popularity votes.
- Spider-Man: No Way Home has highest popularity.
- Year 2020 has maximum movie releases.
